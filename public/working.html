This is a simple chat room web application with two screens: a "Join Chatroom" screen and a "Chat" screen. Join Chatroom Screen: Users enter their username and click "Join" to enter the chatroom. Chat Screen: Once joined, users can see messages from other users, send their own messages, and exit the chatroom. Functionality: The app uses JavaScript and Socket.IO for real-time communication. When a user sends a message, it's displayed in the chat window and sent to the server using Socket.IO. Other users receive the message via the server and it's displayed in their chat window. The "Exit" button disconnects the user from the chatroom.
